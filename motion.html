Motion & Distance Sensors

Motion and distance sensors are used to detect movement, presence, and object distance in IoT and embedded projects.

PIR Motion Sensor

What it is: Detects human motion using infrared radiation.

Working: Senses change in IR levels when a person moves.

Used in:

  • Motion-based lights
  • Intruder alarm systems
  • Smart room automation

Ultrasonic Sensor (HC-SR04)

What it is: Measures distance using ultrasonic sound waves.

Working: Sends ultrasonic pulse and calculates echo time.

Used in:

  • Obstacle detection
  • Smart parking systems
  • Distance measurement projects

IR Obstacle Sensor

What it is: Detects nearby objects using infrared light.

Working: IR light reflects from object and triggers output.

Used in:

  • Line follower robots
  • Object detection
  • Basic robotics

Vibration Sensor

What it is: Detects vibrations or shocks.

Working: Internal spring/contact triggers on vibration.

Used in:

  • Earthquake alert demo
  • Machine vibration monitoring

Tilt Sensor

What it is: Detects inclination or tilt.

Working: Metal ball inside changes contact when tilted.

Used in:

  • Tilt-based alerts
  • Orientation detection

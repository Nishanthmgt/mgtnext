Smart Home using ESP32

Intermediate Level | WiFi-based Home Automation

INTERMEDIATE

Project Overview

This project demonstrates a WiFi-based smart home system using ESP32. Home appliances can be controlled remotely through a web or mobile interface.

Components Required

  • ESP32 Development Board
  • Relay Module
  • AC Appliance (Demo load)
  • Jumper Wires

Pin Configuration

Component ESP32 Pin
Relay IN GPIO 5

System Block Diagram

Communication Flow

  1. ESP32 connects to WiFi
  2. User sends command from mobile/web
  3. ESP32 processes request
  4. Relay switches appliance

Working Principle

ESP32 acts as a web server. When a command is received, it toggles the relay connected to the appliance.

ESP32 Code

#define RELAY 5

void setup() {
  pinMode(RELAY, OUTPUT);
}

void loop() {
  digitalWrite(RELAY, HIGH);
  delay(2000);
  digitalWrite(RELAY, LOW);
  delay(2000);
}

Light & Optical Sensors

Light and optical sensors detect light intensity, color, flame, and infrared signals. They are widely used in automation and IoT projects.

LDR (Light Dependent Resistor)

What it is: A resistor whose resistance changes with light intensity.

Working: Higher light → lower resistance; Arduino reads it as analog value.

Used in:

  • Automatic street lights
  • Smart night lamps
  • Light intensity monitoring

IR Receiver

What it is: Receives infrared signals from remote controls.

Working: Detects modulated IR signals and outputs digital data.

Used in:

  • IR remote control projects
  • Wireless control systems

IR Proximity Sensor

What it is: Detects nearby objects using infrared reflection.

Working: IR light reflects from object and triggers output.

Used in:

  • Object detection
  • Robotics applications

Flame Sensor

What it is: Detects infrared light emitted by fire flames.

Working: Flame IR radiation activates the sensor.

Used in:

  • Fire detection systems
  • Safety alarm projects

Color Sensor (TCS3200)

What it is: Detects RGB color values of objects.

Working: Converts color light into frequency output.

Used in:

  • Color sorting systems
  • Industrial automation demos

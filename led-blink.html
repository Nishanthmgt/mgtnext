LED Blink using Arduino

Beginner Level | Digital Output Basics

Project Overview

This project demonstrates how to control a digital output using Arduino. An LED is turned ON and OFF at regular intervals, helping beginners understand microcontroller pin control.

Components Required

  • Arduino UNO
  • LED
  • 220Ω Resistor
  • Breadboard
  • Jumper Wires

Pin Configuration

Component Arduino Pin
LED Anode (+) D13
LED Cathode (−) GND

Circuit Diagram

Working Principle

Arduino sends HIGH and LOW signals to digital pin D13. When HIGH, the LED turns ON. When LOW, the LED turns OFF. This process repeats continuously, creating a blinking effect.

Arduino Code

int ledPin = 13;

void setup() {
  pinMode(ledPin, OUTPUT);
}

void loop() {
  digitalWrite(ledPin, HIGH);
  delay(1000);
  digitalWrite(ledPin, LOW);
  delay(1000);
}
